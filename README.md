## 폴더 구조

```
C/
├── CodingTest/       # 백준 및 코딩테스트 풀이
├── DataStructure/    # 자료구조 구현
│   ├── List/         # 연결 리스트 (단순/이중)
│   └── Stack/        # 스택 (배열 기반)

└── test.cpp          # 테스트 코드
```
